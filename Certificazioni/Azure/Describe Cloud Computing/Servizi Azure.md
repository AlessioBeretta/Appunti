Le risorse dei servizi che offre Azure tra le più usate sono le seguenti:
- VM: Gestire una VM vera e propria
- Web App: Runnare del codice nel cloud senza gestire la VM
- Function App: Runnare del piccolo codice nel cloud senza avere il server
- Logic App: Consente di collegare due applicazioni per eseguire un processo e, se l'evento si attiva, fare automaticamente qualcos'altro. Per creare un diagramma logico in modo efficace.


## Categorie

### AI

Si hanno molti servizi di AI, quindi ad esempio possiamo passare un testo ad Azure e farlo tradurre dall'AI utilizzando un API e un chat bot.

### Containers

Container sono importanti perchè son dei pacchetti che contengono varie cose e si possono fare anche test sui computer quantistici.

### Database

Gestione dei vari database tramite SQL, Microsoft e anche Oracle. **NB** **Non ci sono solo servizi ufficiali Microsoft ma come in questo caso abbiamo servizi Oracle**

### Marketplace

E' il marketplace in cui altre aziende oltre a Microsoft offrono dei vari servizi anche software Open-source a pagamento.
