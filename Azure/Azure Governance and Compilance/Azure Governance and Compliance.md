
Governance, è una struttura per gestire rischi, conformità e risorse nel cloud Microsoft.

# Tipi di regole:

- Tutti i server devono runnare dei software che sono nelle guide linea di Microsoft Extended Support
- Tutti i server devono avere almeno un back up ogni 24 ore come minimo
- Firewall devono bloccare tutte le porte in ingresso da internet tranne la porta 443
- Solo alcuni individui possono rebootare un server

# Come fare rispettare le regole

Far si che Microsoft controlli le regole che sono state violate o applicare effettivamente le regole.
