# Protocolli SMB o NFS

Il Server Message Block (SMB) è un protocollo di rete client-server utilizzato principalmente per la condivisione di file, stampanti e porte seriali tra nodi di una rete, specialmente in ambienti Windows

"NFS" si riferisce principalmente a: Network File System, un protocollo di condivisione file di rete utilizzato in sistemi Unix/Linux per condividere cartelle su rete.
