# Azure Storage (GPv2)

- "Standard Storage"
-  Suddivisi in quattro tipi di file che può contenere
	-  Container, File, Queue, Table
- Può contenere fino a 5 Petabytes (5PB) = 5 milioni di GB
- Paghi ciò che usi
-  All'incirca 2 centesimi per GB per mese
- Estremamente economico per dello storage in Cloud
- Non raccomandato per ambienti di lavoro ad alta domanda


# Data Lake

- Un impostazione che puoi scegliere quando crei un account storage GPv2
- Un data lake è uno storage estremamente grande
- Può ottenere petabytes e exabytes
-  E' ottimo per l'analisi di "grandi dati"
