Azure Arc consente di gestire VM, server fisici e container al di fuori di Azure come se fossero VM e container di Azure.

# Features

- Le estensioni di  Azure VM permettono ai tool di Azure di lavorare per monitoraggio, sicurezza e aggiornamenti.
- Supporta servizio di dati
- Lavora con i cluster di Kubernetes
- Lavora con Azure Policy
- Management consistent tra i server nell'enviroment.

# Cosa puoi fare?

- Collezionare log per Log Analytics and Monitor
- Usare VM Insights per analizzare le performance
- Scaricare ed eseguire script per connettere macchine ibride
- Aggiornare le certificazioni tramite Key Vault
