## Perchè è importante?

- La gestione delle applicazioni e delle risorse in cloud è molto facile e semplice, ciò che influisce i costi, prestazioni, sicurezza e altre priorità. 
- A seconda del cloud ci saranno differenze in base al fornitore di cloud potrebbe essere piu semplice o difficile


## Come si ottiene?

- Azure Portal, CLI, PowerShell, Cloud Shell, REST APIs e altri metodi programmatici per lavorare con risorse. 
- Monitoraggio e alert di avviso da ricevere una notifica quando le risorse arriva al limite.
- Poter gestire ARM templates, Bicep, Terraform
- Autoscaling che consente di aumentare e ridurre efficacemente l'utilizzo delle risorse in base alle loro esigenze
