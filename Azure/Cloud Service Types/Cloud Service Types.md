## "As a Service"

- Significa che dovremmo comprarlo, ma possiamo affittarlo per tempo, se vogliamo
- Non è necessario iscriversi per tanto tempo, possiamo anche avere un server che si accende alle 8 e spegnere alle 15 e non usarlo mai piu.
- Pagare per cosa usiamo (Tempo, GB, etc..)
- Cloud Provider è responsabile dell'acquisto di server, sviluppo software, manutenzione, aggiornamenti e altri aspetti come abbiamo visto nel SRM


## IaaS

Servizi essenziali della tecnologia:
- Computing
- Storage
- Networking

- Generalmente ci sono degli equivalenti nel mondo reale nel data center.

Quindi si tratta di rimpiazzare in cloud delle cose reali.

## IaaS Computing

- Un esempio è Azure Virtual Machines
- Le VM vengono pagate per secondo
- Scelta in velocità CPU, RAM, e ottimizzazione ecc

## IaaS Storage

- Azure Storage è un esempio di questi
- Offre 5 PB di storage
- Può contenere BLOB, files, code e tabelle di archiviazione.
- Consente la configurazione dell'account per petabyte ed exabyte di dati utilizzando un data lake

## IaaS Networking

- Virtual Networking è un esempio
- Virtual Networking non costa nulla
- Ingress, sono i dati in ingresso da Azure mentre Egress sono i dati in uscita da Azure. E hanno un costo, ma esiste un livello gratuito (5 GB)



## PaaS

- Cloud provider di solito offrono un qualcosa in piu rispetto alla infrastruttura di base
- PaaS include un layer di gestione su IaaS come computing, storage and networking.
- Middleware, strumenti di sviluppo, distribuzione, server di database e altro.

## PaaS Computing

-  Un esempio è Azure App Service
- Carichi il codice e la configurazione su Azure, e runna il codice senza preoccuparti della VM
- Include funzioni di scaling, containers, ambienti di sviluppo e staging

# PaaS Storage

- Managed Storage è un esempio
- Azure SQL Database è un altro esempio di piattaforma
- Non dobbiamo preoccuparci del disco rigido, di quanto è pieno, prestazioni ecc..
## PaaS Networking

- Azure Front Door è un esempio
- Azure Load Balancer
- Azure Firewall
Queste sono le applicazioni che fanno operazioni di networking.

## SaaS

- Cloud Apps
- Come Office 365, OneDrive, Skype
- Le app sono già pronte e dobbiamo solo configurarle e usarle
