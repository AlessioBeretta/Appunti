
Regions:

+ Area del mondo dove Azure ha un set di datacenter (almeno 3)
+ Non devono essere necessariamente in regioni
+ Di solito una regione è collegata a un altra per fare una "region pair"
+ Region pairs hanno la velocità maggiore di connessione e un trattamento speciale durante gli aggiornamenti di Azure.

* Sovereign Azure:
+ Non sono connesse al Azure Public Cloud
+ Serve un approval per entrare / creare una subscription
+ Aderire a diversi standard di conformità e privacy
