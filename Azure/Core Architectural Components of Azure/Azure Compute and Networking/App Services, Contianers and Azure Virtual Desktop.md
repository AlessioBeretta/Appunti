
+ Paradigma per l'esecuzione del codice nel cloud
+ Dando il codice e la configurazione su Azure, lo eseguirà
+ Vedrai una promessa di prestazioni ma non accesso all'hardware
+ Platform as a Service (PaaS)


### Container Services

+ Un altro paradigma per runnare codice nel cloud
+ Containers contengono tutto quello che un app ha bisogno per runnare in "Container Image"
+ Veloce e facile da deployare
+ **Azure Container Istance (ACI)** - Singola istanza, modo piu veloce per deployare un container
+ **Azure Container Apps** - Funzionano come un servizio web, con feature avanzate
+ **Azure Kubernetes Service** (AKS) - Runna su un cluster di server, grado-enterprise


### Azure Virtual Desktop

- Versione Desktop di Windows che runna nel cloud
- Il software è installato, i tuoi file sono disponibili ovunque
- Puoi vedere il tuo desktop su iOS e Android da qualsiasi web browser
- Runna su Azure
